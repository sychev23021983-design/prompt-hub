# Changelog — SEO Content Engine (prompt_config / prompt_templates)

Версионируется контентная часть системы генерации промптов: стиль, лимиты,
чек-лист, формат вывода. Логика данных (Flask/SQLite) версионируется вместе
с остальным кодом репозитория и сюда не включается.

## v1.0 — 2026-07-11
Базовая версия, перенесена из захардкоженного `prompts.py` без изменения
поведения (проверено построчным сравнением вывода для всех трёх сайтов).

- ✓ 3 конфига сайтов: `commerce_en`, `b2b_ru`, `personal_ru`
- ✓ Общий шаблон `master_prompt.md.j2` + общий чек-лист `_shared/quality_check.md.j2`
- ✓ Общие числовые лимиты в `_shared_defaults.yaml` с возможностью override по сайту
- ✓ EEAT-тон, анти-AI-штампы, запрет на выдумывание фактов
- ✓ Локальное SEO (Минск/Беларусь)
- ✓ FAQ, Schema.org, Open Graph, 2-3 изображения с ALT/Title/caption
- ✓ Самопроверка по Yoast перед выдачей ответа

## Как редактировать
- Изменить тон/персону/банворды одного сайта → правьте `prompt_config/<style>.yaml`
- Изменить общий чек-лист или формат вывода для всех сайтов → правьте файлы в
  `prompt_templates/`
- Изменить лимиты (длины полей, кол-во FAQ/картинок) → `prompt_config/_shared_defaults.yaml`,
  либо `overrides:` в конфиге конкретного сайта, если нужно только для него
- Изменения применяются на сервере через `git pull` — пересборка контейнера не нужна
  (код смонтирован как volume)
